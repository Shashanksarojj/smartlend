# SQS + Dead Letter Queue Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace RabbitMQ for loan events with AWS SQS + DLQ via LocalStack, with an `AWS_SQS_ENABLED` toggle so both brokers can be demoed.

**Architecture:** `loan-service` gains a conditional `SqsClient` bean; when `aws.sqs.enabled=true` it serialises event maps to JSON and calls `SqsClient.sendMessage()` instead of `rabbitTemplate.convertAndSend()`. `notification-service` gains a Spring Cloud AWS `@SqsListener` consumer that routes by `type` field to the same existing `LoanNotificationHandler`. The RabbitMQ `@RabbitListener` stays active at all times to handle `user.registered` from user-service (which stays on RabbitMQ). A DLQ (`loan-events-dlq`) receives messages after 3 failed delivery attempts via SQS redrive policy — no application code needed.

**Tech Stack:** AWS SDK v2 `software.amazon.awssdk:sqs:2.25.16`, Spring Cloud AWS 3.1.1 (`io.awspring.cloud:spring-cloud-aws-starter-sqs`), LocalStack at `http://localhost:4566`, Jackson `ObjectMapper` (already on classpath), JUnit 5 + Mockito (existing pattern).

## Global Constraints

- AWS SDK version: `2.25.16` — matches existing `aws.sdk.version` property in `loan-service/pom.xml`
- Spring Cloud AWS version: `3.1.1` — compatible with Spring Boot 3.2.0
- AWS region: `ap-south-1` (Mumbai) — all LocalStack resources
- LocalStack SQS endpoint: `http://localhost:4566`
- LocalStack queue ARN format: `arn:aws:sqs:ap-south-1:000000000000:<queue-name>`
- All new Java: `@Slf4j` (Lombok), `async/await`-equivalent = no sync blocking, package roots `com.smartlend.loan` and `com.smartlend.notification`
- Toggle property key: `aws.sqs.enabled` (maps to env var `AWS_SQS_ENABLED`)
- Queue URL property key: `aws.sqs.queue-url` (maps to env var `AWS_SQS_QUEUE_URL`)
- Existing tests must stay green — do not alter `LoanServiceTest` existing test methods, only add new ones

```java
package com.smartlend.loan.config;

import lombok.extern.slf4j.Slf4j;
import org.springframework.beans.factory.annotation.Value;
import org.springframework.boot.autoconfigure.condition.ConditionalOnProperty;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import software.amazon.awssdk.auth.credentials.AwsBasicCredentials;
import software.amazon.awssdk.auth.credentials.StaticCredentialsProvider;
import software.amazon.awssdk.regions.Region;
import software.amazon.awssdk.services.sqs.SqsClient;
import software.amazon.awssdk.services.sqs.model.CreateQueueRequest;
import software.amazon.awssdk.services.sqs.model.GetQueueUrlRequest;
import software.amazon.awssdk.services.sqs.model.QueueAttributeName;
import software.amazon.awssdk.services.sqs.model.QueueDoesNotExistException;

import java.net.URI;
import java.util.Map;

@Configuration
@Slf4j
public class SqsConfig {

    @Value("${aws.endpoint:}")
    private String endpoint;

    @Value("${aws.region:ap-south-1}")
    private String region;

    @Value("${aws.access-key:test}")
    private String accessKey;

    @Value("${aws.secret-key:test}")
    private String secretKey;

    @Value("${aws.sqs.queue-name:loan-events}")
    private String queueName;

    @Bean
    @ConditionalOnProperty(name = "aws.sqs.enabled", havingValue = "true")
    public SqsClient sqsClient() {
        var credentials = StaticCredentialsProvider.create(
                AwsBasicCredentials.create(accessKey, secretKey));

        var builder = SqsClient.builder()
                .region(Region.of(region))
                .credentialsProvider(credentials);

        if (endpoint != null && !endpoint.isBlank()) {
            builder.endpointOverride(URI.create(endpoint));
            log.info("SQS using LocalStack endpoint: {}", endpoint);
        }

        SqsClient client = builder.build();
        ensureQueuesExist(client);
        return client;
    }

    private void ensureQueuesExist(SqsClient client) {
        try {
            String dlqName = queueName + "-dlq";
            createQueueIfAbsent(client, dlqName, Map.of());

            // LocalStack ARN format: arn:aws:sqs:{region}:000000000000:{queue-name}
            String dlqArn = "arn:aws:sqs:" + region + ":000000000000:" + dlqName;
            String redrivePolicy = "{\"maxReceiveCount\":\"3\",\"deadLetterTargetArn\":\"" + dlqArn + "\"}";
            createQueueIfAbsent(client, queueName,
                    Map.of(QueueAttributeName.REDRIVE_POLICY, redrivePolicy));

            log.info("SQS queues ready: {} → DLQ: {}", queueName, dlqName);
        } catch (Exception e) {
            log.warn("Could not verify/create SQS queues: {}", e.getMessage());
        }
    }

    private void createQueueIfAbsent(SqsClient client, String name, Map<QueueAttributeName, String> attributes) {
        try {
            client.getQueueUrl(GetQueueUrlRequest.builder().queueName(name).build());
        } catch (QueueDoesNotExistException e) {
            var req = CreateQueueRequest.builder().queueName(name);
            if (!attributes.isEmpty()) req.attributes(attributes);
            String url = client.createQueue(req.build()).queueUrl();
            log.info("SQS queue '{}' created: {}", name, url);
        }
    }
}
```

```java
        if (sqsClient != null) {

                String body = objectMapper.writeValueAsString(payload);
                sqsClient.sendMessage(SendMessageRequest.builder()

                        .messageBody(body)
```

```java
package com.smartlend.notification.config;

import lombok.extern.slf4j.Slf4j;
import org.springframework.beans.factory.annotation.Value;

        }
    }

    private void createQueueIfAbsent(SqsClient client, String name, Map<QueueAttributeName, String> attributes) {
        try {
            client.getQueueUrl(GetQueueUrlRequest.builder().queueName(name).build());
        } catch (QueueDoesNotExistException e) {
            var req = CreateQueueRequest.builder().queueName(name);
            if (!attributes.isEmpty()) req.attributes(attributes);
            client.createQueue(req.build());
            log.info("SQS queue '{}' created", name);
        }
    }
}
```

## Task 7: notification-service — SqsConsumerService

**Files:**
- Create: `notification-service/src/main/java/com/smartlend/notification/consumer/SqsConsumerService.java`

**Interfaces:**
- Consumes: `${aws.sqs.queue-url}`, `LoanNotificationHandler` bean (existing)
- Produces: `handleLoanEvent(Map<String, Object> event)` — called by Spring Cloud AWS on message arrival

- [ ] **Step 1: Create SqsConsumerService.java**

```java
package com.smartlend.notification.consumer;

import com.smartlend.notification.handler.LoanNotificationHandler;
import io.awspring.cloud.sqs.annotation.SqsListener;
import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.boot.autoconfigure.condition.ConditionalOnProperty;
import org.springframework.stereotype.Service;

import java.util.Map;

@Service
@ConditionalOnProperty(name = "aws.sqs.enabled", havingValue = "true")
@RequiredArgsConstructor
@Slf4j
public class SqsConsumerService {

    private final LoanNotificationHandler notificationHandler;

    @SqsListener(value = "${aws.sqs.queue-url}")
    public void handleLoanEvent(Map<String, Object> event) {
        String type = (String) event.getOrDefault("type", "");
        log.info("SQS received loan event type={} loanId={}", type, event.get("loanId"));

        switch (type) {
            case "USER_REGISTERED" -> notificationHandler.handleUserRegistered(event);
            case "LOAN_APPLIED"    -> notificationHandler.handleLoanApplied(event);
            case "APPROVED"        -> notificationHandler.handleLoanApproved(event);
            case "REJECTED"        -> notificationHandler.handleLoanRejected(event);
            case "EMI_DUE"         -> notificationHandler.handleEmiDue(event);
            default                -> log.warn("Unknown SQS event type: {}", type);
        }
    }
}
```
